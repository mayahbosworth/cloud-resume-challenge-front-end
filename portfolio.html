Cloud Resume Challenge

(HTML source code)

Key Skills Used:

  • Full-stack web development
  • Cloud services management
  • Secure web protocols
  • Infrastructure as code
  • Continuous integration and delivery
  • Version control and documentation

Tools Used:

    HTML, CSS, JavaScript, AWS, Python, Terraform, GitHub

Project Description:

    Engaged in the Cloud Resume Challenge to develop and demonstrate skills in full-stack software development, cloud services, and CI/CD pipelines. The project encompassed building a cloud-hosted resume website, integrating backend services, and automating deployment processes.

Key Features:

  • Frontend Development:
    • Designed and coded a personal resume using HTML and CSS.
    • Deployed the website as a static S3 site on AWS.
    • Implemented HTTPS using AWS CloudFront for secure access.
    • Configured custom DNS using AWS Route 53 and Namecheap for a personalized domain.
  • Backend API Integration:
    • Developed a visitor counter feature using JavaScript, AWS Lambda (Python), and API Gateway.
    • Set up and managed a NoSQL database (AWS DynamoDB) for storing visitor counts.
  • CI/CD and Automation:
    • Utilized Terraform for infrastructure as code, automating the setup of AWS services.
    • Established source control for backend (Python, Terraform) and frontend (HTML, CSS, JS) using GitHub repositories.
    • Implemented CI/CD pipelines using GitHub Actions for automatic updates upon code changes.

    Outcome:

      This project showcased my ability to integrate a static website with a serverless backend, manage a NoSQL database, and automate deployment processes (and this pretty cool website).

      • Recursive DNS Server
    • Functionality:

      • Purpose: Ensures efficient domain name resolution for client PCs.
    • Authoritative DNS Server and Web Server:
        • Authoritative DNS Server and Web Server:>
